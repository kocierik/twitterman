# TwitterMan

## Run the program

To build and run the program type in the `server` directory:

```
go mod tidy && go run .
```

## Test the program

To test the program type in the directory with some `*_test.go*` file:

```
go test
```

## Run docker

To run dockerfile:

```
docker compose up --build
```
